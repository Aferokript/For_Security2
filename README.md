# Консоль безопасности банка

Система мониторинга пропусков и их визитов сотрудников банка Т-Bank.

## Как установить?

Скачайте репозиторий и сохраните себе его на рабочем столе. Затем в редакторе кода выберите файл с репозиторием.
Python 3.13+ уже должен быть установлен на вашем компьютере.
Вам так же нужны следующие библиотеки из файла requirements.txt:

1. Django - 5.2*
2. psycopg2 binary - 2.9*
3. python dotenv - 0.9.9*

Затем вам нужно настроить данные для подключения в файле .env:

1. DB_HOST=
2. DB_PORT=
3. DB_NAME=
4. DB_USER=
5. DB_PASSWORD=
6. SECRET_KEY=

после всего этого проверьте пути представлений в файле urls.py и можете запускать скрипт.

## Примеры запуска

Вам выдаст список сотрудников из БД, в том числе и их пасскод пропуска.
Кликнув по одному из сотрудников вы увидите все их визиты. Если визит длился больше часа, он становится подозрительным

Так же во вкладке сверху будет информация о струдниках, которые уже находятся в хранилище
